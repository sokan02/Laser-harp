# Laser Harp

DIY laser-based musical instrument built using Arduino.  
Breaking a laser beam triggers a sound or a MIDI note.

## What it does
- Lasers act as “strings”
- Photo resistors detect beam interruption
- Arduino generates sound or MIDI output

## Contents
- harp.ino – Arduino code  
- live_harfa.py – Python MIDI script  
- Schematics.png – wiring schematic

## Technologies
Arduino · Lasers · MIDI · Python

## Getting Started
1. Upload `harp.ino` to the Arduino  
2. Connect components according to the schematic  
3. Run `live_harfa.py` for MIDI output
